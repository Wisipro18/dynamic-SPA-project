# dynamic-SPA

This is my **creative React Single Page Application (SPA)** built for internship submission.

It features:

- Home, About, and Contact pages with smooth navigation (SPA)  
- Dark/Light mode toggle  
- Animated particles background  
- Interactive forms with live validation  
- Smooth page transitions and micro-interactions  
- Theme and user preferences stored across pages
